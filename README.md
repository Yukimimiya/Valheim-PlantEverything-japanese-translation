# Valheim-PlantEverything-japanese-translation
a japanese translation file for PlantEverything

## これは何？
ValheimのMod[PlantEverything](https://www.nexusmods.com/valheim/mods/1042)の日本語翻訳ファイルです。

## 使い方
1. 以下のフォルダを、存在してないなら作成します。
```
Valheim/BepInEx/config/PlantEverything/
```
2. 作成したフォルダ中にファイル`japanese_PlantEverything.json`を置きます。
3. PlantEverythingの設定で、`EnableLocalization`を`true`にし、      Languageを`japanese`にします。

## 注意
- PlantEverythingのバージョン1.20.0の内容で翻訳しています。
- 誤訳には十分注意していますが、きっと存在します、もし見つけたら教えてください。
